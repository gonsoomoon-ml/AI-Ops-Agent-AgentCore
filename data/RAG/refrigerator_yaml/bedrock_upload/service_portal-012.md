# QnA. Service Portal에서 수리 보고서를 잘못 작성하여 수정하고 싶습니다.

> 수리 보고서 수정, 수리 보고서, 제출 후 24시간, 본인 직접 수정, 서비스 현황 메뉴, 수정 버튼, Manager 권한자 승인, 수정 사유, 수정 이력, 서비스 포털

## 관련 질문
- 수리 보고서 수정 알려줘
- 수리 보고서 수정에 대해 설명해줘
- 수리 보고서 수정이 뭐야?
- 수리 보고서 수정 뭐야?
- 수리 보고서 수정 어떻게 해?
- 수리 보고서 수정 알려주세요
- 수리 보고서 수정이 뭔가요?
- 수리 보고서 수정 무엇인가요?

## 답변

수리 보고서 수정은 제출 후 24시간 이내에는 본인이 직접 수정할 수 있습니다. 1. 서비스 현황 메뉴에서 해당 서비스 건을 선택합니다. 2. 수리 보고서 하단의 수정 버튼을 선택합니다. 3. 변경할 내용을 수정하고 다시 제출합니다. 24시간이 경과한 경우에는 Manager 권한자의 승인이 필요합니다. Manager에게 수정 사유를 전달하고 승인을 받으면 수정이 가능합니다. 보고서 삭제는 지원하지 않으며, 수정 이력은 시스템에 기록됩니다.
