# 냉장고 문제 발생 시 에러 코드 확인 방법

> 에러 코드 확인, 제어 패널, 디스플레이, 깜빡이면서, 푸시 알림, 재부팅, 전원을 뽑고, 에러 이력, 22E, 5E, 8E, 진단/에러코드

## 관련 질문
- 에러 코드 확인 알려줘
- 에러 코드 확인에 대해 설명해줘
- 에러 코드 확인이 뭐야?
- 에러 코드 확인 뭐야?
- 에러 코드 확인 어떻게 해?
- 에러 코드 확인 알려주세요
- 에러 코드 확인이 뭔가요?
- 에러 코드 확인 무엇인가요?

## 답변

냉장고에 문제가 발생하면 제어 패널 디스플레이에 에러 코드가 표시됩니다. 에러 코드를 확인하는 방법은 다음과 같습니다. 1. 제어 패널에서 직접 확인: 디스플레이에 숫자와 알파벳 조합의 에러 코드(예: 5E, 8E, 22E)가 깜빡이면서 표시됩니다. 정상 작동 시에는 온도가 표시되지만 에러 발생 시 에러 코드로 전환됩니다. 2. SmartThings 앱에서 확인: 에러 발생 시 SmartThings 앱으로 푸시 알림이 전송되며, 앱에서 에러 코드와 함께 간단한 조치 방법이 안내됩니다. 3. 서비스 모드에서 확인: 서비스 기사는 서비스 모드 진입 후 전체 에러 이력을 확인할 수 있습니다(서비스 모드 진입 방법은 별도 항목 참조). 에러 코드가 표시되면 우선 냉장고 전원을 뽑고 1분 후 다시 꽂아 재부팅해 보시기 바랍니다. 재부팅 후에도 에러가 지속되면 서비스 센터에 문의하시기 바랍니다.
