# Service Portal에서 제품 등록 방법

> 제품 등록, 모델번호, 시리얼번호, 구매일자, 명판, 병행수입, 영수증, 고객정보, 서비스 포털

## 관련 질문
- 제품 등록 알려줘
- 제품 등록에 대해 설명해줘
- 제품 등록이 뭐야?
- 제품 등록 뭐야?
- 제품 등록 어떻게 해?
- 제품 등록 알려주세요
- 제품 등록이 뭔가요?
- 제품 등록 무엇인가요?

## 답변

Service Portal에서 고객의 냉장고를 등록하는 방법은 다음과 같습니다. 1. Service Portal에 로그인 후 제품 등록 메뉴를 선택합니다. 2. 제품 정보를 입력합니다. - 모델번호: 냉장고 내부 또는 뒷면 명판에 표기된 모델번호 입력 - 시리얼번호: 명판에 표기된 시리얼번호 입력 - 구매일자: 고객의 구매 영수증 기준 일자 입력 3. 고객 정보(이름, 연락처, 주소)를 입력합니다. 4. 등록 완료 버튼을 누르면 제품이 시스템에 등록됩니다. 제품 등록 시 모델번호가 시스템에 존재하지 않는 경우, 병행수입 제품이거나 모델번호 입력 오류일 수 있으므로 명판 정보를 다시 확인하시기 바랍니다.
