## 답변

삼성 냉장고의 모델번호는 다음과 같은 체계로 구성되어 있습니다. 예시: RF85A9121AP 1. 제품 유형 코드 (1~2자리): - RF: 프렌치도어(French Door) - RS: 양문형(Side-by-Side) - RT: 상냉장 하냉동(Top Mount Freezer) - RB: 하냉동(Bottom Mount Freezer) - RR: 1도어 냉장전용 - RZ: 1도어 냉동전용 - RQ: 4도어(T-Type) 2. 용량 코드 (2자리): 총 용량을 10L 단위로 표기 (예: 85 = 850L급) 3. 디자인/연도 코드 (1자리): 알파벳으로 출시 연도와 디자인 세대 구분 4. 기능 등급 코드 (1자리): 숫자가 높을수록 상위 기능 탑재 5. 색상/사양 코드 (2~3자리): 도어 색상, 패널 재질 등을 코드로 표현 6. 지역 코드 (2자리): 판매 지역을 나타냄 (예: AP=아시아태평양, SR=한국)
