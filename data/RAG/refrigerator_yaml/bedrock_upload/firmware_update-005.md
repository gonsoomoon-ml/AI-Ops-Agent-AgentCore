## 답변

냉장고 펌웨어 업데이트가 실패한 경우 다음과 같이 대응할 수 있습니다. 1. Wi-Fi 연결 상태 확인: 업데이트 중 Wi-Fi 연결이 끊어졌을 수 있으므로 연결 상태를 확인하고 재연결합니다. 2. 재시도: SmartThings 앱에서 다시 업데이트를 시도합니다. 다운로드가 완료된 상태에서 설치에 실패한 경우 자동으로 재시도됩니다. 3. 전원 재부팅: 냉장고 전원 플러그를 뽑고 1분 후 다시 꽂아 재부팅한 후 업데이트를 재시도합니다. 4. 네트워크 초기화: 냉장고의 Wi-Fi 설정을 초기화하고 재연결한 후 업데이트를 시도합니다. 업데이트 실패가 반복되면 서비스 센터에 문의하시기 바랍니다. 업데이트 실패 시에도 기존 펌웨어는 유지되므로 냉장고의 냉각 기능에는 영향이 없습니다.
