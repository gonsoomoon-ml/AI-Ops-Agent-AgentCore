## 답변

냉장고 펌웨어 업데이트는 Wi-Fi 연결 상태에서 다음과 같은 시점에 업데이트를 확인합니다. 1. 최초 Wi-Fi 연결 시: SmartThings 앱에 냉장고를 등록하고 Wi-Fi에 연결하면 즉시 업데이트 가능 여부를 확인합니다. 2. 주기적 확인(Polling): 1일에 1회 자동으로 서버에 접속하여 새로운 펌웨어가 있는지 확인합니다. 확인 시점은 새벽 시간대(02:00~05:00)에 랜덤으로 설정됩니다. 3. 수동 확인: SmartThings 앱의 설정 > 정보 메뉴에서 사용자가 직접 업데이트 확인 버튼을 눌러 확인할 수 있습니다. 4. 네트워크 재연결 시: Wi-Fi 연결이 끊어졌다가 다시 연결되면 업데이트 가능 여부를 확인합니다.
